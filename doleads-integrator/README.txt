DoLeads Integrator plugin connects your wordpress website contact form with 'DoLeads' Leads Management System.

== Description ==

DoLeads Integrator plugin connects your wordpress website contact form with 'DoLeads' Leads Management System.

DoLeads Integrator provides the ability to Wordpress admin to bind any DoLeads Groups with any form of the supported types. Once form fields are mapped with DoLeads group fields, whenever the form is submitted, the same will also be submitted to DoLeads as per the field mapping.

No programming knowledge is required. Anyone who knows the DoLeads groups that he/she wants to bind can do the integration with WP forms. 


A few notes about the sections above:

* Easy setup and integration
* Store data from your WordPress contact plugin in DoLeads
* Simple to use UI to map the form fields with the DoLeads group fields.
* Keep the data map even if you deactivate the plugin


### Compatibility
DoLeads Integrator plugin currently works with the following plugins;

Open source plugins

* [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) Supported Version(s): 4.6+

== Installation ==

#### Automatic Installation
1. Visit Plugins > Add New
2. Search for DoLeads Integrator
3. Activate DoLeads Integrator from your Plugins page.

#### Manual Installation
1. Download DoLeads Integrator Plugin.
2. Upload the ledo-integrator directory to your /wp-content/plugins/ directory, using your favorite method (ftp, sftp, scp)
3. Activate DoLeads Integrator from your Plugins page.

#### Manual Installation
1. Download DoLeads Integrator Plugin.
2. Visit Plugins > Add New
3. Click on upload plugin button.
4. Click on browse button and select plugin zip file. After file select click on Install Now button.
5. Click on Activate Plugin button.

== Privacy ==

This plugin submits the contact form information such as name, telephone number, email address etc. of your website visitors to our third party service [DoLeads - Leads Management System](https://leads.docloud.global) for marketing purposes when they submits their contact information on your website.

We use the information we collect in various ways, including to:

* Communicate with your website visitors, either directly or through one of our partners for marketing and promotional purposes.
* Send emails to your website visitors.

For more information please read our [privacy policy](https://leads.docloud.site/privacy-policy.php)
